# Hitos y Tareas para Construir el Testimonials Slider

## Hitos

1. **Inicialización del Proyecto**
    - Crear un repositorio público en GitHub.
    - Configurar el repositorio para publicar el código en una dirección web.

2. **Estructuración del Contenido**
    - Analizar los diseños en la carpeta `/design`.
    - Planificar la estructura HTML del proyecto.

3. **Estilos Base**
    - Definir los estilos base en CSS, incluyendo `font-family` y `font-size`.

4. **Desarrollo del Slider**
    - Implementar la estructura HTML del slider.
    - Aplicar estilos CSS para el diseño del slider.
    - Añadir funcionalidad con JavaScript para la navegación del slider.

5. **Optimización para Dispositivos**
    - Asegurar que el diseño sea responsive y se vea bien en diferentes tamaños de pantalla.

6. **Pruebas y Ajustes**
    - Probar la funcionalidad del slider con mouse/trackpad y teclado.
    - Realizar ajustes según sea necesario para cumplir con el diseño.

7. **Despliegue del Proyecto**
    - Publicar el proyecto en una plataforma de hosting (GitHub Pages, Vercel, Netlify).

8. **Documentación y Entrega**
    - Crear un `README.md` personalizado.
    - Subir la solución a la plataforma de Frontend Mentor.
